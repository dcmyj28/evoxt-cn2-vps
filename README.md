## CN2 线路到底是什么？为什么值得专门去找？

很多人买 VPS 的时候只盯着 CPU、内存、带宽，然后买回来发现访问速度一塌糊涂。问题通常不在配置，而在线路。

普通的国际 BGP 线路，数据包从大陆出去之后，可能要绕一大圈才能到目标服务器，中间经过的节点多、拥堵点多，晚高峰时段尤其明显。CN2 线路不一样，它是中国电信的"高速公路"，节点少、路径短、QoS 优先级更高，所以延迟更稳定，丢包率也低很多。

**CN2 线路有两种：**

- **CN2 GT（Global Transit）**：轻度优化，高峰期偶尔会出现一点拥堵，价格相对便宜。
- **CN2 GIA（Global Internet Access）**：顶级精品线路，专线级别，全程没有公共拥堵点，延迟最低、最稳定，价格通常也更贵。

对于大多数用户来说，有 CN2 回程优化就已经够用了；追求极致低延迟的，可以专门找 CN2 GIA 的方案。
